Hi there Year 12s! Have you ever wondered just how to use that piece of hardware? This helpful guide is going to show you not only how to use a series of excellent hardware devices but also how they work! Did you know that some keyboards use a silicone membrane as a spring to push those keys back up? Find out this and many more facts in this helpful guide.

Mouse

What it Does

A mouse translates movement along a surface to movement of the cursor on a computer. An older style mechanical mouse uses a ball which rolls against a series of rollers inside the mouse to detect the direction the mouse is being moved. An optical mouse uses an LED or laser which works with a light sensor and microprocessor. The LED or laser produces light which bounces off the surface. The reflection is detected by the light sensor and the micoprocessor analyses the created image. Changes in the images over time is converted very reliably into movement of the cursor on the screen, translating the direction and speed of the mouse movement. the buttons on the mouse are simple electronic switches that register the clicks and send the data to the computer. The scroll-wheel is a potentiometer which changes the resistance in a circuit based on the change of position. All of this data is sent to the computer (through wire or wireless receiver) which is deciphered via the driver files by the operating system.

How to Use

A mouse allows you to interact with programs on the computer by moving a pointer around the screen and click on items to select or choose from a menu. The main affordances of a mouse are moving the pointer, left and right clicking and scrolling. The pointer is a little icon that moves around the screen, changing its look based on the available interactions. It can be moved by sliding the mouse around the surface. if you find that you get to the edge of the surface before reaching the edge of the screen the mouse can be lifted and repositioned. A mouse typically has two or three buttons left and right click and sometimes a scroll button. The left button is used to select, click on hyperlinks, drag and drop and do the most common interactions on the computer. the right click is used for extra options that are contextual for what you are clicking on. The third button is often to be found behind the scroll on the mouse. It is used to use the mouses movement to scroll the page rather than the scroll wheel - it is an accessibility option. Scrolling up or down a page, like on a web page or text document, is done by rolling the scroll-wheel in the middle. There is a possibility of extra buttons found on the sides of a mouse. Often the functionality of these buttons is able to be set by the user.

Keyboard

What it Does

A keyboard is one of the most essential input devices to a computer. It was the first and remains the most useful method of interfacing with your computer. It is made up of many small keys that send a signal to the computer when depressed. The most commonly used keys are the alphabet characters and space. The keys are laid out in a specific pattern called "English US QWERTY" (named for the first 6 letter keys). There are also a series of other characters and number as well as some action keys. All of these keys have a specific code as defined by the American Standard Code for Information Interchange.

There are many different ways that a keyboard can be engineered. One of the most common is a membrane keyboard using a dome switch. Dome switch keyboards use plastic domes to suspend the keys above a circuit board. When the key is depressed the plastic dome is flattened onto the board creating a circuit with conductive material on the underside of the dome. This sends a signal to the processor on the keyboard which converts it to the binary string to send to the computer. With combinations of keys the action key can alter the binary string of the letter key. An example of this is the shift key which alters one bit in the letter binary to switch the capitalisation.

How to Use

There are many types of keys on a keyboard, the most commn type are the letter keys. These keys work in a simple way, inputting the labled letter, which can be capitalised by pressing them in combination with additional keys like Shift or Capslock. In some software programs, these may not type letters, but instead may perform an action. An example of this is how keys like W, A, S, and D keys are used instead to move a character around the screen in many games. Similar to these is the Space key. This key is used to input the space character when inputting letters to form words, or some other actions in different software programs.

Then there are the special characters and number keys. All of these have two lables per key, the bottom one being the main character, and the higher one being what characcter is sent when combined with the Shift key. An example of this is the ` key, which becomes the ~ key when combined with shift. Another type of key is the Action key, which alter the affect of a key, like the Ctrl key, which can be used to do commands such as Ctrl+s command, which saves a document. another type of key on a keyboard are the Function keys, labled as f1, f2 and so on. These can be used for computer-specific commands, for example changing the volume. the last type of key is an arrow key. these are labled with arrows, and are used for text navigation mostly, but some programs can use them for different, usually movement related actions.

Speaker

What it Does

The main parts of a speaker are the cone and dust cap: which move the air arount to create sound vibrations, the spider and the surround, which hold the cone in place while allowing it to move, the magnet and voice coil: which turn electricity into movement to move the cone, the basket, the pole and top plate, and the Frame. the way the voice coil works is using the fact that when electricity is run through a coil it produces a magnetic field. The magnetic field moves the coil towards or away from a magnet, depending on its polarisation. When audio signals are sent through the coil, it is attracted and repelled by the permanent magnet on the speaker, depending on the height and polarity of the wave.

Sound files are stored in digital formats, while sound waves themselves are in analog form. The speaker needs to use a Digital Analog Converter (DAC) to turn the digital wave into an analog wave for the sound creation.

How to Use

Speakers are fairly simple items to use, but there are many types. The most common type would be a speaker that is embedded inside a device, like a Laptop or a phone. for this kind of speaker, nothing excess is needed to be done to play audio, simply press the play button. Another common type of speaker is found in headphones and earbuds. for these to work, you must connect your device to them. Two common ways of doing this are with an audio jack, and an aux cable or with Bluetooth. another type is an external speaker, which like headphones or earbuds can be connected to using an aux cable or Bluetooyth, but these often need their own power source, which is a trait not oftenfoundin wired headphones/earbuds. one last type of speaker is of that found in an amp. Amps need to be plugged into an outlet to function, and are most often co0nnected to an instrument or microphone directly using a guitar or mic lead.

LCD Data Projector

What it Does

An LCD Data Projector is the main type of projector in modern use, using a single light bulb. the light projected from this lightbulb is seperated into red green and blue using Dichronic mirrors, the coloured light is then reflected into LCD screens which map the respective Red green and blue values from each pixel, and the light is then recombined with a prism.

Data projectors, like regular computer displays, create images using pixels that each have their own RGB values, which represent a colour, for example RGB 0,0,0 is black, RGB 255,0,0 is Red, and an rgb value of 252,3,186 is a nice shade of pink. there are heaps of pixels that make up a single image, for example, a regular 4K screen is 3840 pixels by 2160 pixels. That's 8294400 total pixels!

How to Use

There are a couple ways to use a data projector, depending on the projector itself, your computer and what cables you have at hand. As our need for quality of video has progressed, so has the ways we interface between the laptop and the display. because of this, the cables we use have evolved with us. The first cable used for transmitting video was the composite video cable. this compressed video data into a single cable, with a single pin, sending a single stream of data, with a maximum size of 720 × 576i ar 50 Frames per second(FPS). This worked completely fine for the standards of the time it was introduced, but if you look back on anything using this cable from the perspective of today's technology, it looks terrible. there have been many additions and improvements to video cables over time, but not many have stood the test of time. Until recently with the Fad of reducing the amount of ports to just one type, requiring dongles to be able to do anything whatsoever, the two most used video cables were VGA, with a maximum resolution of 2048 × 1536 at 85FPS and the more common HDMI, with a huge maximum of 10240 x 4320 pixels at 120 frames per second.
